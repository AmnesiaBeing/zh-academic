# zh-academic - Typora 中文 LaTeX 学术风格主题

此主题衍生于官方 [Academic 主题](https://theme.typora.io/theme/Academic/)，中文使用华文中宋。

## 安装方法

直接下载本仓库为 ZIP，将内容（`ZH-academic/` 和 `zh-academic.css`）解压缩至 Typora 主题文件夹内。
